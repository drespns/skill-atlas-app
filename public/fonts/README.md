## Fonts (self-host)

Este proyecto soporta presets de fuente en `Ajustes` (`skillatlas_prefs_v1` → `font`).

Si quieres añadir tipografías **self-host**, coloca aquí los `.woff2` y define sus `@font-face` en `src/styles/fonts.css`.

Recomendación:
- Solo `.woff2`
- 1–2 pesos por familia (regular + semibold) para no inflar el bundle
